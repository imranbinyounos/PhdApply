# GradKit Pro - User Guide

## Getting Started with GradKit Pro

### Quick Start Tutorial

#### 1. First Launch
When you first open GradKit Pro, you'll see the Dashboard which provides an overview of your PhD application progress. The interface is designed to be intuitive, but here's a quick tour:

- **Navigation Bar**: Switch between Dashboard, Universities, Professors, and Scholarships
- **Main Content Area**: View and manage your data
- **Sidebar**: Quick filters and search options (when available)

#### 2. Adding Your First Professor
1. Click on the **Professors** tab
2. Click the **"+" button** or **"Add Professor"**
3. Fill in the required information:
   - Professor Name
   - Email Address
   - University
   - Department
   - Research Interests (detailed description)
4. Set initial status (usually "Researching")
5. Add any initial notes
6. Click **Save**

#### 3. Managing Application Status
Each professor entry can be tracked through various stages:
- **Researching**: Initial investigation phase
- **Drafting Email**: Preparing initial contact
- **Contacted**: Email sent, awaiting response
- **Awaiting Response**: Following up on communication
- **Interview Scheduled**: Meeting arranged
- **Submitted**: Formal application submitted
- **Accepted/Rejected**: Final decision received

#### 4. Adding Universities and Programs
1. Navigate to the **Universities** tab
2. Add comprehensive university information:
   - University name and location
   - Department details
   - Program requirements
   - Application deadlines
   - Contact information

#### 5. Tracking Scholarships and Funding
1. Use the **Scholarships** tab to track funding opportunities
2. Include application deadlines and requirements
3. Link scholarships to specific universities or programs

## Workflow Recommendations

### Optimal Application Timeline
1. **3-6 months before deadlines**: Begin professor research
2. **2-3 months before**: Initial email contact
3. **1-2 months before**: Follow up and submit applications
4. **After submission**: Track responses and decisions

### Research Organization Tips
- **Use priority levels** to focus on best-fit opportunities
- **Maintain detailed notes** on research alignment
- **Track all interactions** for reference
- **Set realistic deadlines** with buffer time

### Email Management Best Practices
- **Log all communications** with dates and summaries
- **Use templates** for consistency but personalize each message
- **Set follow-up reminders** to maintain momentum
- **Include relevant links** to your work and interests

## Troubleshooting Common Issues

### Data Entry Problems
- **Invalid Email Formats**: Ensure proper email syntax
- **Date Format Issues**: Use system date picker for consistency
- **Large Text Fields**: Use the notes section for extensive information
- **Link Validation**: Verify URLs are complete and accessible

### Performance Optimization
- **Large Datasets**: Enable performance mode in preferences
- **Memory Usage**: Restart app if interface becomes sluggish
- **Search Speed**: Use specific terms rather than broad searches
- **Export Operations**: Be patient with large CSV exports

### Import/Export Issues
- **CSV Formatting**: Ensure proper column headers and data format
- **Character Encoding**: Save CSV files with UTF-8 encoding
- **Date Formats**: Use ISO format (YYYY-MM-DD) for best compatibility
- **Special Characters**: Ensure proper handling of international characters

## Keyboard Shortcuts

### Navigation
- **⌘1**: Dashboard
- **⌘2**: Universities
- **⌘3**: Professors  
- **⌘4**: Scholarships
- **⌘F**: Search
- **⌘N**: New Record
- **⌘S**: Save Current Record

### Data Management
- **⌘A**: Select All
- **⌘C**: Copy Selected
- **⌘V**: Paste
- **⌘Z**: Undo
- **⌘⇧Z**: Redo
- **⌘E**: Export Data
- **⌘I**: Import Data

### View Options
- **⌘+**: Increase Text Size
- **⌘-**: Decrease Text Size
- **⌘0**: Reset Text Size
- **⌘⇧D**: Toggle Dark Mode
- **⌘R**: Refresh View
